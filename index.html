    ellar Portal API

    The Stellar Portal network coordinates constellations, wormholes, and expedition schedules across the deep-space relay grid. Use this static page when publishing to GitHub Pages for a fully client-side Swagger UI experience.

    Base URL: https://api.stellarportal.example.com/v1

    Authentication

    All write operations require an X-Stellar-Key header issued by Mission Control. Requests originating from observatory nodes must also include X-Relay-ID so telemetry can be correlated.

    Key

    Entities
    • Constellations — curated star groups with visibility windows.
    • Wormholes — transient travel corridors ranked by stability.
    • Relays — orbital infrastructure that broadcasts navigation signals.

      • Operational Notes

      Sandbox keys are rate limited to 60 requests per minute. Production keys inherit relay-level burst quotas.
